# Smart Time Tracker

A simple web app with:

- Stopwatch
- Multiple countdown timers
- User name input
- Delete timer feature

## Features

- Start / Stop / Reset stopwatch
- Add multiple timers
- Delete any timer
- Mobile friendly

## Technologies

- HTML
- CSS
- JavaScript

## How to Run

1. Download project
2. Open `index.html` in browser
